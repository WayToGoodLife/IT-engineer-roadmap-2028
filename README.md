## 📚 Структура каждого этапа

Каждый этап состоит из недель. Каждая неделя включает:

1. **Цели недели** — что вы изучите
2. **Важно перед стартом** — чек-лист готовности
3. **Напутствие** — мотивация и фокус
4. **Ресурсы на неделю** — ссылки на курсы, видео, документацию
5. **Задачи по дням** (🟢 день недели, ⭐ приоритет, [ ] чек-бокс, время)
6. **Чек-лист завершения недели** — что должно быть сделано
7. **Советы после недели** — как закрепить

---

## ⚠️ Ключевое правило

**Каждый день вы должны сделать хотя бы критические задачи (⭐).** Базовые задачи (•) — если остаётся время.

Если вы сделали только ⭐ — это уже хороший день. Прогресс не остановился.
