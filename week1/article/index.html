Что такое HTML и зачем он нужен

HTML - это язык разметки. С его помощью мы говорим браузеру: это заголовок, это абзац, это список.

Без HTML страница была бы просто сплошным текстом, как в блокноте.

Мои любимые семантические теги

  1. header — для шапки сайта
  2. nav — для меню
  3. article — для отдельной статьи или поста
  4. footer — для подвала

Цитата дня

Код, который работает — это хорошо. Код, который понимает другой человек — это профессионализм
